# Sunad OTT — Documentation Set (Phase 1)

**Documentation-First Engineering — before any UI design begins.**

This set documents Sunad Broadcast Pvt. Ltd.'s OTT platform: a premium, bilingual (Hindi/English) streaming home for Indian civilizational storytelling — spirituality, history, documentary, journalism, tourism, and literature — designed with the restraint of Apple, the discovery mechanics of Netflix, and the quiet luxury of Aman Resorts. It is explicitly **not** a religious, temple, or mythology platform.

## How to use the "Assumption" labels

Because Sunad's own websites (`sunad.in`, `mysticpower.in`) currently expose very little crawlable content (see `01_Report.md` §0 and §7), most company-specific specifics in this set are reasoned from the brief plus market research, not pulled from verified company data. Every such claim is explicitly marked **[Assumption]**. Before design work begins, these should be reviewed with Sunad stakeholders — see the **Assumptions Log** in `01_Report.md` §11 for the consolidated list of open questions.

## The one rule that governs every document here

*Restraint reads as trust. Hindi is designed, not translated. When in doubt, remove — and check every decision against the brief's explicit non-goals: not religious, not temple, not mythology.*
